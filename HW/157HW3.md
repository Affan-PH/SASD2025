1) Utility Class

Class: System.Math  
Namespace: System

Description:
System.Math เป็นคลาสที่รวมเมธอดสำหรับคำนวณทางคณิตศาสตร์ เช่น รากที่สอง ยกกำลัง และค่าตรีโกณมิติ

Example:
```
using System;

class Program
{
    static void Main()
    {
        double result = Math.Sqrt(25);
        Console.WriteLine(result);
    }
}
```
วิเคราะห์ System.Math เป็น static class ไม่สามารถสร้าง object ด้วย new ได้
ทุกเมธอดเป็น static และไม่มีการเก็บ state ภายใน
จึงจัดเป็น Utility Class เพราะเป็นคลาสเครื่องมือสำหรับเรียกใช้งานฟังก์ชันโดยตรง

2) Mutable Class

Class: System.Text.StringBuilder
Namespace: System.Text

Description:
StringBuilder ใช้จัดการข้อความที่ต้องมีการแก้ไขหลายครั้งโดยไม่ต้องสร้าง object ใหม่ทุกครั้ง

Example:
```
using System;
using System.Text;

class Program
{
    static void Main()
    {
        StringBuilder sb = new StringBuilder("Hello");
        sb.Append(" World");
        Console.WriteLine(sb.ToString());
    }
}
```
วิเคราะห์ เมื่อเรียกใช้ Append() ค่าภายใน object เดิมจะถูกแก้ไข
ไม่มีการสร้าง instance ใหม่ทุกครั้งที่เปลี่ยนค่า
ดังนั้น StringBuilder เป็น Mutable Class เพราะ state ภายในสามารถเปลี่ยนแปลงได้

3) Immutable Class

Class: System.String
Namespace: System

Description:
String เป็นคลาสสำหรับเก็บข้อมูลประเภทข้อความ และใช้งานบ่อยในภาษา C#

Example:
```
using System;

class Program
{
    static void Main()
    {
        string s = "Hello";
        s = s + " World";
        Console.WriteLine(s);
    }
}
```
วิเคราะห์ เมื่อมีการต่อข้อความด้วย + จะเกิด object ใหม่ขึ้นมา
ค่าเดิมใน memory ไม่ถูกแก้ไข
จึงจัดเป็น Immutable Class เพราะไม่สามารถเปลี่ยนแปลงค่าเดิมได้
